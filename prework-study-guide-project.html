HTML

  • HTML is made up of elements that perfrom distinct roles.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata is data about the webpage that the browser uses to give information about the webpage to the users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declaration blocks contain two important components: Property and Value.
  • Do Not Repeat Yourself(DRY): a coding principle that encourages developers to reuse or share pieces of code.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit: Record changes to the repository.
  • git switch: Switch branches.
  • git pull: Fetch from and integrate with another repository or a local branch.
  • git init: Create an empty Git repository or reinitialize an existing one.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Variables that are surrounded by quotation marks(") are refered to as Strings.
  • A function is simply a set of directions that tell the computer how to perform a task.
  • In JavaScript, for a function to work, you need a keyword and then call the function.
